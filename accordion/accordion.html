Accordion Demo

If you're using Wordpress, a custom parent theme has been created to allow for full implementation of the byu components. This theme includes easy backend input for both the header and footer, and allows for complete customization of site content. A child theme has also been developed and made available as well, for those who would prefer to customize their header and footer display for their department. Developed by Publications & Graphics.

The Drupal Bootstrap theme is based off of the Bootstrap theme, giving you the flexibility and default styling that comes from Twitter's Bootstrap. It also can use modules developed specifically for that theme and most of the styling grunt work has already been done for you. If you use this theme, it's important to know that it does not use the components, it just recreates them. So, you'll need to be sure you keep the theme update so it matches with the components. Developed by SAAS.

This is a Drupal sub-theme based on the ZURB Foundation theme. It includes a grid system as well as options to easily edit BYU header features, fonts, and font colors. This theme makes it very easy to customize how your content is displayed with not fear of messing up the components. Also, because this theme pulls in the components from the web community, you're site will always have the latest component updates. Developed by OIT.
